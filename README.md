# NeuroNotes
NeuroNotes is a full-stack web application that helps students upload, organize, and access study materials such as notes, PPTs, and videos. Developed using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite, it provides authentication, subject-based browsing, and file management.
The system also allows students to share their own resources, creating a collaborative environment where study materials can be exchanged within the student community.

✨ Features

User Signup and Login authentication

Personal dashboard for users

Upload study materials (Notes, PPTs, YouTube links)

Browse resources by subject

Download or view uploaded files

Delete uploaded resources

Search subjects easily

Curated educational links for important subjects

Getting Started
1️⃣ Clone the repository
git clone https://github.com/hemanthghk24/neurornotes.git
cd neurornotes
2️⃣ Install dependencies
npm install
3️⃣ Start the backend server
node server.js

The backend will run on:

http://localhost:4001
4️⃣ Run the frontend

Open index.html in your browser
(or run using VS Code Live Server).
